# Hello OpenGL

> A simple hello world program for OpenGL beginners.

## How to Run Program in Terminal

1. Clone repo
2. Inside project folder, run `make` command
3. Type `./hello`

## What it Looks Like

You should see an image of a yellow triangle with a black background.
